программа com_grabber ловит данные с ком-порта в кольцевой буфер и сохраняет его в файл out.txt
Опции:
-p "port", номер порта
-b "baud rate", скороть подключения
-s "size", размер буфера (в байтах) = размер файла out.txt = сколько данных програ может поймать с ком-порта
-m "mode", формат сообщения (байты данных + проверка четности + стоп-бит)
-l "loop", записывать всё время в out.txt по кругу или записать один раз и выйти
-a "append", присоединить новые данные к существующему файлу (имеет смысл использовать только если включен флаг loop)
-w "wait", время (в мс), которое подождать, прежде чем забрать данные из ком-порта
-p5 -b9600 -s256 -m8N1 -l1 -w3000
